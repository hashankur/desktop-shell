## Dependencies

```sh
# Bundler
ags

# Clipboard
wl-clipboard
cliphist
imagemagick

# OSD
brightnessctl

# Color generation (optional)
matugen
```

## Prerequisities

Install npm packages (use preferred package manager):

```sh
bun install
```

Generate types:

```sh
ags types
```

## Start Desktop Shell

```sh
ags run --gtk4
```
